Welcome to Kangmo's F/A-18C Manual Pages

Purpose of These Pages

These pages are specifically designed for Microsoft Edge's read-aloud feature. Unlike PDF files where Edge stops reading after the text visible on screen, these HTML pages allow continuous reading - perfect for listening to F/A-18C manuals while riding a bike or running for an hour or more.

Each page includes high-quality images (300 DPI) of the original PDF pages, allowing you to view the original manual content in detail while also benefiting from the continuous read-aloud functionality.

The pages are structured to announce the current page number out of total pages (e.g., "Page 42 of 300"), helping you keep track of your progress through the manual as you listen.

These pages have been tested on iOS devices using the Microsoft Edge browser and should work equally well on Android devices with the Edge browser installed.

To use this feature, open these pages in Microsoft Edge and activate the read-aloud feature. Learn more about Edge's read-aloud feature here.
